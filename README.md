# Retail Sales SQL Analysis




## 📌 Project Overview
This project demonstrates SQL skills by analyzing a retail sales dataset using MySQL. It covers database creation, data understanding, data cleaning, exploratory data analysis (EDA), and advanced SQL concepts to generate meaningful business insights.



## 🛠️ Tools Used
- MySQL Workbench
- SQL
- GitHub



## 📂 Project Structure

- 01_database_setup.sql
- 02_data_understanding.sql
- 03_data_cleaning.sql
- 04_exploratory_data_analysis.sql
- 05_advanced_sql.sql
- SampleSuperstore.csv
